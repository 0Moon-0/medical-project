## ✨ Features

### 👨‍⚕️ Doctor Panel
- View and manage **Medical Files**
- Manage patient **Analysis** results
- View **Radiation** records
- Create and view **Reports**
- Manage **Endoscopies**
- Full **CRUD** operations (Add, Delete, Update, Search)

### 🧑‍⚕️ Patient Panel
- View personal **Medical File**
- Access **Analysis** results
- View **Radiation** history
- Read **Doctor Reports**
- Full **CRUD** operations

### 🔐 Authentication
- Separate login for Doctors and Patients
- Registration with role selection (Doctor / Donor / Recipient)
- Remember Me functionality
